=================
django-cryptocoin
=================

Django-cryptocoin is a Django app to organize accepting bitcoin, litecoin, novacoin and other cryptocoins.

Quick Start
===========

1. Install using pip:

        pip install django-cryptocoin

2. Add to INSTALLED_APPS in your `settings.py`:

        'django-cryptocoin',
